# 人脸识别
*2019.8.6更新*      
&emsp;&emsp;通过opencv做人脸识别    
&emsp;&emsp;代码实现了通过电脑的摄像头自动检测视频中的人脸、眼睛、微笑。

- *不足之处*    
&emsp;&emsp;1.当人脸在视频中是斜的时检测不到    
&emsp;&emsp;2.误识别率高
